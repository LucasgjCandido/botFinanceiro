# Bot Financeiro WhatsApp

Bot pessoal para controle de gastos diretamente pelo WhatsApp. Permite registrar despesas rapidamente por mensagem e consultar resumos financeiros do dia ou do mês.

## Funcionalidades

- Registro de gastos por mensagem
- Resumo do dia
- Resumo do mês
- Histórico de gastos
- Exclusão do último gasto
- Cancelamento por código

## Tecnologias utilizadas

- Node.js
- Express
- WhatsApp Cloud API
- OpenAI GPT-4o-mini
- SQLite
